  • While completing assignment one, index.html, I had trouble with Git and Github. Pushing commits not only to my developent machine but syncing them to my Github repository was confusing at first because I was under the impression that the repositories were automatically linked.
  • In order to understand this better I used github docs and practiced pushing commits to ensure my files were saving to the repo properly.
  • This assignment introduced me to working with Git/Github and the basic version control etiquette that goes with it. I was already familiar with the introductory formatting in html such as unordered lists and headers.
  • The github docs mentioned earlier and w3schools were the resources that enabled me to complete assignment 1.
  • I used the gitHub doc to learn how to push commits and I used w3schools to learn more html syntax, like an img tag.
